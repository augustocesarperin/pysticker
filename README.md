### Como Executar

**Requisito:** Python 3

1.  **Clone o repositório:**
    ```sh
    git clone https://github.com/augustocesarperin/pysticker.git
    ```
2.  **Navegue até a pasta:**
    ```sh
    cd pysticker
    ```
3.  **Execute o aplicativo:**
    ```sh
    python main.py
    ```
